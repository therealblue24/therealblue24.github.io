Fun (TM) with C

Note: very simplified

The beginning

Ok, so I was experimenting with pointers during my free time and found something interesting.

I could modify a variable using another variable. You may already know where I'm going with this.

"Memory Safe", more like a joke

Let's take this program:


#include <stdio.h>

int main() {
    int a = 1;
    int b = 3;
    int c = 4;
    int d = {2, 5};
    printf("a = %d, b = %d, c = %d, d = {%d, %d}\n", a, b, c, d[0], d[1]);
    return 0;
}

It's just a toy program that makes variable a, b, and c and makes a array d.

And in C, variables first are in the stack top.

So our memory looks like this:


bottom                 top
+----+----+----+----+----+
| d0 | d1 | c  | b  | a  |
+----+----+----+----+----+

Now, for the trolling:

Our objective is to change A through B.

How would we do this? Well, we can refrence B to get a pointer:



                   &b
                    v
+----+----+----+----+----+
| d0 | d1 | c  | b  | a  |
+----+----+----+----+----+

We can then add b by 1 to get a pointer to A.



                   &b+1
                     v
+----+----+----+----+----+
| d0 | d1 | c  | b  | a  |
+----+----+----+----+----+

Then we can derefence our pointer to get A. Now, we can do something like this:


*((&b)+1) = 13; /* Oh no! A is now 13! */

This kids, is the wonders and fun of C.
